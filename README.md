# LisezMoiCreator

Générateur de fichier lisez-moi pour la confrérie des traducteurs

## Description

Application moderne en Python avec interface graphique PyQt6 pour créer facilement des fichiers README structurés.

## Prérequis

- Python 3.8 ou supérieur
- PyQt6

## Installation

1. Cloner le dépôt
2. Installer les dépendances :
   ```bash
   pip install -r requirements.txt
   ```

## Utilisation

Lancer l'application :
```bash
python main.py
```

L'interface permet de remplir différentes sections du README :
- Informations générales (titre, auteur, version, date, résumé)
- Prérequis
- Description détaillée
- Instructions d'installation
- Instructions de désinstallation
- Crédits et remerciements

## Historique

Initialement écrit en Visual Basic, puis porté en Qt/C++, l'application a été modernisée en Python pour une meilleure maintenabilité et portabilité.
